# Социальная сеть EveOnline (Alpha release)
## Модули: 
- Парсер EVE Swagger
- Приложение для мультиязычного перевода
- Web UI приложение
- Web API приложения
- Юнит Тестирование
- Доступы по внутрикорпоративным ролям
## Стек технологий
- C#, Asp.Net Core, Identity, WebAPI, Entity Framework Core, Async-await
- Unit Tests, Functional testing
- Базы данных MSSQL и PostgreSQL
- Брокер сообщений RabbitMQ
- Распределенное кэширование Redis
- SignalR
- WebUI, JQuery, CSS + SASS
- Docker, Kubernetes
- Gulp, Webpack
